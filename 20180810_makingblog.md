## 왜 jekyll?

- 가입형 블로그 거른다
- 동적 페이지 블로그 거른다
- GitHub Pages 로 호스팅 하기 편해야 한다
  - [GitHub Pages](https://pages.github.com/)
- [블로그 프레임워크 비교, 왜 지킬인가?](https://qvil.github.io/blog/why-jekyll/)
